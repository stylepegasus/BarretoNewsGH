# Banco de Imagens - Barreto News

Este é o local correto para salvar todas as imagens do site (Logos, ícones, banners, favicons).

## Como adicionar uma imagem:

1.  Cole o arquivo de imagem aqui nesta pasta.
2.  Renomeie para um nome simples, sem espaços e acentos.
    *   ✅ `logo-principal.png`
    *   ❌ `Logo do Site Versão Final (2).png`
3.  Avise ao Antigravity o nome do arquivo.

## Exemplo de uso no código:

Se você salvou o arquivo como `foto-hero.jpg` aqui, nós usaremos assim no código:

```jsx
<img src="/images/foto-hero.jpg" alt="Descrição" />
```

## Arquivos sugeridos para começar:

*   `favicon.png` (O ícone que fica na aba do navegador)
*   `logo.png`
*   `og-image.jpg` (Imagem que aparece quando compartilha o link no WhatsApp)
